Farming Products(React-Redux Product Inventory System Use Case)
	Farming Products app is used for managing the stocks of the products.
	Main functionalitys are,
		1. Product Display (Used for displaying all products you can edit,delete product here)
		2. DashBoard (Used for stockmanagement and displays the charts for sale rate) 

<---  Installation and Setup Instructions  --->

1)Clone down this repository. You will need node and npm installed globally on your machine.
2)Installation:
	npm install
3)To Start react and json Server:
	npm run launch

To Visit App:
	http://localhost:4000/
